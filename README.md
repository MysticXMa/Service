# Screen Share Application

A real-time screen sharing application using a central server.

## Features
- Share your screen with others
- View others' screens
- Secure connection with passwords
- Real-time updates

## Installation
1. Install requirements: `pip install -r requirements.txt`
2. Run: `python screen_share.py`

## Usage
- Enter your name and password
- Click "Start Sharing" to host
- Or "View Available Servers" to connect to others
